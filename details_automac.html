automac

automac is a software that can controll macOS application with voice commands. This software runs in the background and always keeps listening to the user. But only with the wake-up command and with the help of speech recognition module of python and google's gTTS this program can:

1. open/close application
2. switch between applications
3. take screenshot
4. handle brightness of screen and manage volume level of speakers.

Tech Stacks:
  • Python
  • Speech Recognition, gTTS
  • playaudio, playsound
  • AppleScript
